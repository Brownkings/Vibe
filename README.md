# Yedibe News Platform

A modern, conservative South African news media platform built with Vite, Node.js, Firebase, and Cloudinary.

## Features
- **Articles**: Read the latest news in Politics, Society, and more.
- **Videos**: Watch multimedia reports.
- **Admin Dashboard**: secure content management system.
